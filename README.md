# File Renamer
A desktop app for Linux and Windows for batch renaming files.
It's Free, Libre, Open Source Software (FLOSS).

## Locations

1. [Github](https://github.com/flossapps/file-renamer/)
1. [PyPi](https://pypi.org/project/file-renamer/)
